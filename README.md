Halfhoun (Simply Plural Alternative)

<h2>If you don't know what you're doing in code, read this next bit, otherwise, try it out. It's fun!</h2>

Please make sure you have python installed, and you create a fresh virtual environment. There aren't many dependencies, (pygame), but best practices!

This was originally designed to be a discrete, visual way to demonstrate your plurality, without calling anything into question. This is the first iteration.

This simulates the exterior portal design I had been working on. In the code, go down to line 203, and you can change the colors and names to fit your alters. The line will look like this

```
alters = [ {"name": "Princess",  "color": (255, 0, 119),   "host": True},
        {"name": "Oren",    "color": (122, 119, 120),   "host": False},
        {"name": "Kaylee",   "color": (96, 191, 191),   "host": False},
        {"name": "Solara",   "color": (189, 21, 21),   "host": False},
        {"name": "Count",  "color": (137, 173, 99),   "host": False},
]
```

Change the names to your names, and the colors to whatever colors you like. 

In your virtual environment, you run ```pip install -r requirements.txt``` and it'll install what's needed. Then hit play!
